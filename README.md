# 🚀 React Native Navigation (Expo)

Proyecto de práctica enfocado en navegación en React Native con Expo Router, implementando:

- 📚 Stack Navigation
- 📑 Tabs Navigation
- 📑 Drawer Navigation
- 🧭 Estructura basada en rutas (file-based routing)
- 🎨 Estilos con NativeWind

## 🔧 Instalación

1. **Clona el repositorio:**

```bash
git clone https://github.com/anibalcoder/expo-navigation.git
```

2. **Navega a la carpeta del proyecto:**

```bash
cd expo-navigation
```

3. **Instala las dependencias:**

```bash
pnpm install
```

5. **Ejecuta el servidor de desarrollo:**

```bash
pnpm start
```
